Anima

Animation classes are writed with 3 parts :

  • the prefix part is ever "ani"
  • the main part is the animation name, concat if necessary with each animation option
  • the suffix part is one of the six states ("it", "all", "on", "in", "each" or "many")

PS: All examples of animation syntax are writed with "it" because this is the state for regular animation (onload), but for the demonstration, the used state is "in".

Animation Name

All states must be concat with name & prefix, like this :

  • for default animation on the element itself, put this class on it :

    class="ani-grow-it"

  • for default animation to all children, put this class on the parent :

    class="ani-grow-all"

  • for hover/focus animation on the element itself, put this class on it :

    class="ani-grow-on"

  • for hover/focus the parent then animate one child, put this class on the child :

    class="ani-grow-in"

  • for hover/focus animation to each child, put this class on the parent :

    class="ani-grow-each"

  • for hover/focus the parent then animate all children, put this class on the parent :

    class="ani-grow-many"
